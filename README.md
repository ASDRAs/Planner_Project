# 🟣 NEON GENESIS ARCHIVE

> **"100% AI-Driven Vibe Coding Project"**

이 프로젝트는 개발자의 직관(Vibe)과 AI의 기술력을 결합하여 완성된 **개인용 스마트 플래너**입니다. 코드 한 줄 한 줄을 직접 타이핑하는 대신, AI 에이전트와의 긴밀한 협업(바이브 코딩)을 통해 아키텍처 설계부터 UI 세부 디테일, 성능 최적화까지 모든 과정이 구현되었습니다.

**⚠️ 이 README 문서 또한 AI에 의해 작성되었습니다.**

---

## 🤖 Vibe Coding Methodology
- **Pure Conversation**: 모든 기능 구현, 버그 수정, 성능 최적화는 AI 에이전트와의 대화만으로 진행되었습니다.
- **Spec-First & TDD**: AI가 스스로 테스트 코드를 작성하고 이를 통과시키며(Red-Green-Refactor) 안정성을 확보했습니다.
- **Zero Boilerplate**: 반복적인 설정 작업을 AI에 위임하여 오직 '기능'과 '경험'에만 집중했습니다.

## ✨ Key Intelligence
- **MAGI Auto-Classification**: Gemini API를 사용하여 입력된 자연어를 분석하고, 할 일(TODO)과 지식(Archive)을 지능적으로 분리합니다.
- **Neural Sync**: 로컬 스토리지를 기반으로 하며, Supabase를 통해 기기 간 실시간 동기화를 지원합니다.
- **PWA Optimized**: 별도의 설치 없이 브라우저를 통해 데스크탑과 모바일에서 네이티브 앱처럼 작동합니다.

## 🛠 Pilot Guide (How to Use)
입력창에 무엇이든 적으세요. 시스템이 문맥을 이해하고 배치합니다.
- **형식 자동 인식**: `카테고리/폴더/내용` 또는 `내일 할 일`처럼 일상 언어로 입력하면 자동으로 분류됩니다.
- **지식 아카이빙**: 긴 텍스트나 학습 데이터는 'Archive Base'에서 카드 형태로 관리 및 확장이 가능합니다.
